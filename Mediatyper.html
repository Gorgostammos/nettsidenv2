Mediatyper

Media typer i css kan bli forklart med at det er regler som du setter inn på css for å gjøre forksjellige og spesifikke ting i nettsiden din. Eksempler kan var at skjærmen skal var 500px bred. Forskjellige stiler i medier er veldig fornuftig å bruke siden da kan du f.eks. endre på bakgrunnsfargen til det du ønsker .Det finnes mange forskjellige mediatyper vi har i css som kan da være iblant de her:

  • Screen Screen kan bli brukt for å fokusere på fargeskjermer.
  • TV Blir brukt for å fokusere på visninger for tv'er
  • Braille Braille blir brukt for å fokusere på bildeskrift
  • Projection Projection blir brukt for å hjelpe til med prosjektorer.

    • Media typer er bra for å få mer fleksibilitet også spesifisere på hva vi ønsker å gjøre det bedre på, men det kan også være en ulempe med at mediatyper er begrenset for de ulike mediatypene og det kan være svært dekket av responsive design. Greia var at det var ikke brukt mye før. Dette her ble ikke brukt så mye før, så det vi har nå gjør at vi har mange muligheter til å se på hva vi gjør med de media typene, og det gjør at det ikke er mange begrensinger som kan da være bra og ikke bra. ink til det jeg fikk informasjon fra: W3.org
